Invoice Verification Assistant

請求書を自動検証し、検証結果をダッシュボードで提示する React + FastAPI アプリケーション

ウィザードステップ

要件入力
設計プレビュー
UI モック承認
テンプレート生成
バックエンド設定
テスト準備
成果物ダウンロード

検証ルール

  • 請求書番号の重複チェック
  • 金額 1,000,000 円以上の支払いを危険度 "Critical" としてフラグ
  • ベンダーごとのクレジットスコア参照

生成成果物

  • React フロントエンド (MUI Stepper)
  • FastAPI バックエンド + Pydantic モデル
  • Playwright E2E テスト + Faker テストデータ
  • Dockerfile / docker-compose / logging 設定
